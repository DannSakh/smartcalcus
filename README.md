# SmartCalcus — DEV-площадка (статический прототип)

Это минимальный прототип сайта с одним рабочим калькулятором (кредитный), чтобы быстро показать структуру и UX.

## Как развернуть
1) Залейте все файлы на любой хостинг (в корень сайта или поддомен `staging.smartcalcus.online`).  
2) Убедитесь, что включён HTTPS (Let's Encrypt).  
3) Откройте `index.html` и проверьте навигацию.

## Файлы
- `index.html` — главная
- `calc-credit.html` — кредитный калькулятор (аннуитет/дифф), график
- `assets/style.css` — стили
- `assets/app.js` — логика калькулятора
- `sitemap.xml`, `robots.txt` — SEO основы
- `README.md` — этот файл
